随机提示词与随机宏:
  随机提示词:
    “突然出现了一群强盗，人数是 {{number1}} 人，他们的战斗力是 {{number2}}。”
    (假设你设了 number1 是 1-5，number2 是 1-100)
  插件生成时的操作：
    先抽中这句话（随机提示词的功能）。
    再摇骰子把里面的数字填上（随机数宏的功能）。
  AI 最终看到的：
    “突然出现了一群强盗，人数是 3 人，他们的战斗力是 99。”
